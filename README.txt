1.- Luego de tener instalado NODE JS, en la ruta de nuestro proyecto ejecutar el comando para generar un proyecto de react:

npx create-react-app react-nav   

2.- instalar route react para las rutas:
npm install react-router-dom@6

3.- Instalar Bootstrap para react 
 npm install react-bootstrap bootstrap

 4.- Dentro de nuestro archivo App js, borrar el contenido q esta dentro del DIV  y generar nuestras rutas

 5.-Crear nuestra carpetas components y layouts

------- Ejecutar el proyecto con-----
  npm start
